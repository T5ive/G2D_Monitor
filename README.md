# 鹅鸭杀辅助工具
支持玩家信息(位置、是否可见、是否进雾等)、游戏信息（当前地图、房间号、当前游戏进程等）读取

使用Mono原生函数读取地址，游戏更新基本不用更新代码

仅供学习交流，不对使用后果负责，本项目代码不对游戏数值进行修改，如要修改需同时修改隐藏的加密数值，详情请参考http://codestage.net

# 计划
添加每轮的状态分析，包括玩家是否开刀、是否进雾等

记录玩家死亡时附近的玩家

添加玩家位置的地图标记

添加每轮游戏回放功能!
